Installation:

1) run ant (in Eclipse) and make dist
1.1) for configuring ant, add Environment variable (External Tools Conf -> Environment)
     -> Key: JAVA6_BOOT_CLASSPATH
     -> Val: /usr/lib/jvm/java-6-openjdk/jre/lib/rt.jar

2) install the files:
cd ~/Documents/temp
R CMD INSTALL extraTrees

3) package extraTrees:
R CMD build extraTrees
